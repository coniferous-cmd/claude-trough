# Roadmap

## v0.1

Implemented core task management:

- Add tasks.
- List tasks.
- Toggle completion.
- Delete tasks.
- Store tasks in SQLite.
- Edit task details with an external editor.
- Provide a terminal list UI.

## v0.2

Planned or partially introduced capabilities:

- Priority support.
- Search.
- Tags.
- Better task filtering.
- Clearer command semantics for `done` and `undo`.

## Future Ideas

- Explicit complete/uncomplete database operations.
- Priority validation.
- Detail preview or detail pane in TUI.
- Import/export support.
- More complete documentation and examples.

Keep future work aligned with the single-binary, local-first, terminal-first direction.
